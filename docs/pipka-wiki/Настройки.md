#config

# Настройки

Все настройки — через `.env` файл на сервере (`/opt/pipka/.env`).
Модель: `app/config.py` (pydantic-settings).

## Обязательные

| Переменная | Описание |
|-----------|---------|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота |
| `ADZUNA_APP_ID` | Adzuna API ID |
| `ADZUNA_APP_KEY` | Adzuna API Key |
| `ANTHROPIC_API_KEY` | Claude API Key |
| `POSTGRES_PASSWORD` | Пароль PostgreSQL |
| `GOOGLE_CLIENT_ID` | Google OAuth Client ID |
| `GOOGLE_CLIENT_SECRET` | Google OAuth Client Secret |
| `SESSION_SECRET` | Секрет для подписи session cookies |

## Опциональные

### База данных
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `DATABASE_URL` | `sqlite+aiosqlite:///./pipka.db` | Переопределяется docker-compose |

### Google OAuth
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `ADMIN_EMAILS` | `""` | Comma-separated emails → роль admin |

### Скоринг (#scoring)
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `MAX_JOBS_PER_SCORING_BATCH` | `8` | Вакансий за один запрос к Claude |
| `MAX_SCORED_PER_SEARCH` | `30` | Макс скоринга за поисковый запрос |
| `SCORE_CACHE_HOURS` | `168` | 7 дней кэш оценок |
| `CLAUDE_TIMEOUT_SECONDS` | `60.0` | Таймаут запроса к Claude |
| `CLAUDE_MAX_RETRIES` | `2` | Повторные попытки |
| `CLAUDE_MODEL` | `claude-sonnet-4-20250514` | Модель Claude |
| `CLAUDE_SCORING_MAX_TOKENS` | `5000` | Токены для батч-скоринга |
| `CLAUDE_ANALYSIS_MAX_TOKENS` | `1500` | Токены для детального анализа |

### Очистка
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `JOB_MAX_AGE_DAYS` | `45` | Возраст вакансий для удаления |

### Источники
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `ARBEITSAGENTUR_API_KEY` | `jobboerse-jobsuche` | Bundesagentur für Arbeit (публичный) |
| `JOOBLE_API_KEY` | `""` | Jooble meta-aggregator (лимит 500 req/ключ) |

### Gemini Flash (бесплатный backfill скоринг)
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `GEMINI_API_KEY` | `""` | Google AI Studio API key. Пусто → backfill идёт через Claude |
| `GEMINI_MODEL` | `gemini-2.0-flash-lite` | Модель Gemini |
| `GEMINI_BATCH_DELAY` | `4.0` | Задержка между батчами (сек), чтобы не превысить 30 RPM |

### NVIDIA Build (idle rescorer для DE)
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `NVIDIA_API_KEY` | `""` | build.nvidia.com ключ. Пусто → `_nvidia_idle_rescore` no-op |
| `NVIDIA_MODEL` | `google/gemma-4-31b-it` | Модель NVIDIA Build |
| `NVIDIA_BASE_URL` | `https://integrate.api.nvidia.com/v1` | Endpoint (OpenAI-совместимый) |
| `NVIDIA_BATCH_DELAY` | `2.0` | Задержка между вызовами (сек) |
| `NVIDIA_MAX_PER_RUN` | `300` | Cap вакансий за один тик scheduler |
| `NVIDIA_COUNTRY` | `de` | ISO-код страны для фильтра idle rescore |
| `NVIDIA_RESCORE_STALE_DAYS` | `7` | Освежать оценки старше N дней |

### Бэкапы БД (Backblaze B2, опционально)
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `B2_KEY_ID` | `""` | Backblaze B2 key ID (оставить пустым = только локальный бэкап) |
| `B2_APP_KEY` | `""` | Backblaze B2 application key |
| `B2_BUCKET` | `""` | Имя B2 bucket (напр. `pipka-backups`) |
| `B2_ENDPOINT` | `https://s3.us-west-004.backblazeb2.com` | S3-совместимый endpoint B2 |

Локальный бэкап всегда запускается в `02:30 UTC`. Файлы: `/app/data/backups/` (последние 7).

### Sentry (опционально)
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `SENTRY_DSN` | `""` | DSN с sentry.io. Пусто → SDK не инициализируется, нулевая нагрузка. |
| `SENTRY_ENVIRONMENT` | `production` | Тег окружения в Sentry UI |
| `SENTRY_TRACES_SAMPLE_RATE` | `0.05` | Доля запросов с performance-traces |
| `SENTRY_PROFILES_SAMPLE_RATE` | `0.05` | Доля профилирования |

### Прочее
| Переменная | Дефолт | Описание |
|-----------|--------|---------|
| `DEFAULT_RESULTS_LIMIT` | `50` | Вакансий на страницу по умолчанию |

## Настройки профиля пользователя (UI)

Хранятся в [[База данных#user_profiles]]. Управляются через Settings tab на сайте.

| Поле | Описание |
|------|---------|
| Resume text | Текст резюме / ключевые навыки |
| Target titles | Желаемые должности |
| Target Companies Watchlist | Список компаний для мониторинга (Nestlé, Bayer…) — скан каждые 6 ч |
| Excluded keywords | Слова-исключения (не показывать вакансии с ними) |
| **English Only** | ✅ Только вакансии с английским как рабочим языком |
| Min salary | Минимальная зарплата EUR/год |
| Experience years | Лет опыта |
| Languages | en:C1, de:B1 и т.д. |
| Work mode | Any / Remote / Hybrid / On-site |
| Preferred countries | Страны поиска |
| Industries | Отрасли |

> `Base location` удалён (апрель 2026) — не влиял на результаты.

## UI / Dashboard

| Функция | Описание |
|--------|---------|
| Dark mode | Кнопка 🌙/☀️ в шапке. Сохраняется в `localStorage`. Автодетект системной темы. |
| Language switcher | EN / RU / DE / ES кнопки в шапке. Полная локализация UI через `TRANSLATIONS` JS-объект + `data-i18n` атрибуты. Дефолт: RU. |
| Source filter | Adzuna, Indeed, LinkedIn, Glassdoor, Xing, Arbeitsagentur, Arbeitnow, Remotive, ⭐ Watchlist |
| Watchlist badge | Вакансии от компаний из Watchlist подсвечиваются золотым бэджем `⭐ watchlist` |
| Ops tab | Исключительно на русском языке (хардкод, не i18n). Показывает метрики по пользователям/клиентам. |
