# База данных

PostgreSQL 16. Все таблицы создаются через SQLAlchemy `Base.metadata.create_all` при старте. Мигации — мягкие (`ALTER TABLE ... IF NOT EXISTS`), каждая в отдельной транзакции.

## Схема связей

```
users ──< user_profiles
users ──< job_scores ──> jobs
users ──< applications ──> jobs
users ──< search_subscriptions
```

---

## Таблица: `users`

| Колонка | Тип | Описание |
|---------|-----|----------|

**Индексы:** `ix_users_email` (partial, WHERE email IS NOT NULL), `ix_users_google_sub` (partial)

## Автоочистка

Вакансии старше **45 дней** удаляются ежедневно в 03:00 UTC.
Исключение: вакансии со статусом `applied` / `interviewing` / `offer` — хранятся бессрочно.
