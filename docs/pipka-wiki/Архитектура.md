## Поток данных: Сканирование вакансий

```
APScheduler (каждые 3ч)
    │
    ▼
_background_scan()
    │
    ├─► JobAggregator.search()
    │       ├─ AdzunaSource
    │       ├─ JobSpySource (LinkedIn, Indeed, Glassdoor)
    │       ├─ ArbeitnowSource
    │       └─ RemotiveSource
    │
    ├─► Дедупликация (dedup_hash)
    ├─► Фильтр: _is_negative(), _is_wrong_location()
    ├─► Upsert в jobs таблицу
    │
    └─► Для каждого пользователя:
            ├─ pre_filter() — быстрая проверка без AI
            ├─ score_jobs() — Claude AI батчами по 8
            └─ Telegram push если score ≥ 80
```

## Поток данных: Авторизация

```
Browser → /auth/google/login
    → Google OAuth2 (authlib)
    → /auth/google/callback
    → get_or_create_google_user()
    → request.session["user_id"] (signed cookie pipka_session)
    → Redirect → /
```
